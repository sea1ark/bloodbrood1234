# bloodbrood

Простой форум. Написан на PHP без фреймворков.

## Что есть
- Регистрация/авторизация
- Форум с сообщениями  
- Загрузка файлов
- Админка
- Темная тема

## Установка
1. Закинуть файлы на сервер
2. Настроить БД в config/database.php
3. Запустить setup.php
4. Готово

## Требования
- PHP 7.4+
- MySQL
- Apache с mod_rewrite
